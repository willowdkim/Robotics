## Goal
This repository is part of my long-term goal of building robotics systems for real-world impact.

I’m using Webots (a realistic simulation environment) to develop skills in
- robot control
- system modeling
- bridging simulation to realworld systems
- learning and applying concepts through hands on work

All progress is documented through screen recordings and experiments.

## What this represents

Rather than a collection of finished projects, this is a record of
- how I approach problems
- how systems evolve over time
- how I debug and iterate

## Direction

I’m especially interested in
- MedTech robotics
- human-interaction systems
- simulation and digital twin environments

This repository is an early step toward building systems in these areas.
